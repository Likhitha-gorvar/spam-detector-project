# SMS Spam Classifier

This project uses machine learning to classify SMS messages as spam or ham (not spam).

- **File:** `spam.csv`
- **Columns:**
  - `v1`: label (spam/ham)
  - `v2`: message text

- Preprocessing with NLTK
- TF-IDF vectorization
- Naive Bayes classification
- Evaluation using precision, recall, F1-score
